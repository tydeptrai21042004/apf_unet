# Deprecated APF guide

The former APF proposal has been replaced by the plain Fourier U-Net design.
Use [`FOURIER_UNET_GUIDE.md`](FOURIER_UNET_GUIDE.md) and the canonical model key
`proposal_fourier_unet` for new experiments.

Backward-compatible model aliases remain available for old notebooks, but they
must not be used as separate methods in new result tables.
